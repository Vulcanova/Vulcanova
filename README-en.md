# Vulcanova
An unofficial Vulcan UONET+ mobile cross-platform register client made in Xamarin. Currently, in heavy development.

## What works:
- lucky number
- grades
- timetable
- attendance

## What's planned:
- messages 
- dashboard
- homework
- exams
- notes (points)
